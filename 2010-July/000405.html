Hello!

Since version 0.10.7 Smb4K works with recent versions of Samba (i.e. with the 
cifs-utils package). The only thing you might need to do is to add yourself to 
the /etc/sudoers file through Smb4K's configuration dialog (Super User page). 
Previous versions won't work, since they search for umount.cifs which does not 
exist anymore.

Best regards
Alexander
