Hello!

First the easy part: Smb4K is a KDE program, but it works also within GNOME. 
However, it will use KDE applications e.g. to show the contents of the 
shares, etc.

Now to your problem: I never experienced the error message you reported, so I 
had to google for an answer. I found several threads that discuss the 
problem. As far as I can see, the error message NT_STATUS_DUPLICATE_NAME 
might be caused either because you have two servers with the same name or one 
server with two IP addresses. Is that the case?

Here are two threads that might be of some help:
http://www.linuxquestions.org/questions/linux-networking-3/smbclient-cannot-connect-200970/
http://www.backupcentral.com/phpBB2/two-way-mirrors-of-external-mailing-lists-3/backuppc-21/new-member-92158/

By the way, have you configured Samba?
