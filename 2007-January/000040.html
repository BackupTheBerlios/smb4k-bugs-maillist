For Ubuntu Dapper, you can find newer versions (0.8.0) on our FTP site. Add the following
lines to your /etc/apt/sources.list:

deb ftp://ftp.berlios.de/pub/smb4k/ubuntu/dapper/ ./
deb-src ftp://ftp.berlios.de/pub/smb4k/ubuntu/dapper/ ./

Please note that the package located there is not an official one but packaged by me. So, 
bug reports should be filed to the Smb4K-general mailing list:

https://lists.berlios.de/mailman/listinfo/smb4k-general

If the bug persists with the latest version, please notify me.
